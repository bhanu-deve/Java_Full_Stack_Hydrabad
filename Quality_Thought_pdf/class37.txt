Q) Write a java program to display the string character by character?

input:
	hello

output:
	h
	e
	l
	l
	o

ex:
---
class Test  
{
	public static void main(String[] args) 
	{
		String str="hello";
		
		for(int i=0;i<str.length();i++)
		{
			char ch = str.charAt(i);
			System.out.println(ch);
		}
	}
}

Q) Write a java program to count number of digits , perform sum of digits and check it is even or odd?

input:
	NEX123

output:
	Number of digits : 3
	Sum of digits : 6
	It is even number  

ex:
---
class Test  
{
	public static void main(String[] args) 
	{
		String str="NEX123";
		
		String newStr = str.replaceAll("[^0-9]","");
		
		System.out.println("No of Digits : "+newStr.length());
		
		int sum=0;
		for(int i=0;i<newStr.length();i++)
		{
			int n = Character.getNumericValue(newStr.charAt(i));
			sum+=n;
		}
		System.out.println("Sum of Digits : "+sum);
		
		if(sum%2==0)
			System.out.println("It is even number");
		else
			System.out.println("It is odd number");
	}
}

Q) Write a java program to convert uppercase letters to lowercase letters and vice versa?

input:
	IhUBTalEnT 

output:
	iHubtALeNt 

ex:
---

class Test  
{
	public static void main(String[] args) 
	{
		String str="IhUBTalEnT";
		
		char[] carr = str.toCharArray();
		
		for(int i=0;i<carr.length;i++)
		{
			if(carr[i]>='A' && carr[i]<='Z')
				carr[i] = (char)(carr[i]+32);
			else if(carr[i]>='a' && carr[i]<='z')
				carr[i] = (char)(carr[i]-32);
		}
		
		//store the data in string 
		str="";
		for(char c : carr)
		{
			str+=c;
		}
		System.out.println(str);
	}
}
